Design Components

The reusable building blocks behind RL1 Weapon Audit. Each component is shown standalone with its markup — copy any block and drop it into your own project. Include styles.css (or just the relevant rules) and you're set.

Color tokens

:root vars · per theme

All colors flow from CSS variables on :root. Switching data-theme on the app root remaps every token. The accent gold is the only saturated color in the palette; everything else is a tinted neutral.

Typography

cormorant · inter tight · jetbrains mono

A serif/sans/mono triad. The serif (Cormorant Garamond) carries titles and item names. The sans (Inter Tight) is body and UI. The mono (JetBrains Mono) is for stat labels, requirement values, and any "system" text — small, all-caps, wide letter-spacing.

Buttons

.btn-primary · .btn-ghost

Primary uses the gold gradient and uppercase mono label — reserved for the dominant action. Ghost is a hairline outline that picks up the accent on hover.

html
<button class="btn-primary">Auto-solve</button>
<button class="btn-primary" disabled>No solution</button>
<button class="btn-ghost">Clear</button>

Slot card

.slot · empty & filled

The core inventory tile. Bracketed corner serifs on every state, dashed border when empty, solid when filled. The icon lozenge swaps shape per slot kind: circle = talisman, teardrop = tear, diamond = armor.

html · empty
<button class="slot" data-empty="1" data-kind="talisman">
  <div class="slot-frame">
    <div class="slot-corner tl"></div><div class="slot-corner tr"></div>
    <div class="slot-corner bl"></div><div class="slot-corner br"></div>
    <div class="slot-empty">
      <div class="slot-icon" data-kind="talisman"></div>
      <div class="slot-empty-label">Talisman</div>
      <div class="slot-empty-hint">tap to equip</div>
    </div>
  </div>
</button>

Weapon header / picker

.weapon-picker · .weapon-bar

A single underlined line: small mono category label, large serif name, dropdown caret. The simplest possible "select" pattern, with a popover dropdown that includes search and a 3-column row layout.

Picker modal

.picker (inline preview)

A list-of-items modal used for slot equipping. Each row is a self-contained mini-card: name, stat delta, italic flavor. Shown here inline (not as an overlay) so the structure is visible.
